# Clim.io

A React + TypeScript application for optimizing investment portfolios based on ESG (Environmental, Social, Governance) metrics and carbon ratings.

## Features

- View and analyze a 10-stock portfolio with ESG and carbon metrics
- Find and compare peer companies with better sustainability ratings
- Replace holdings with more sustainable alternatives
- Export optimized portfolio as CSV
- Real-time portfolio metrics comparison

## Tech Stack

- React 18 with TypeScript
- Vite for build tooling
- Tailwind CSS for styling
- Functional components with React Hooks
